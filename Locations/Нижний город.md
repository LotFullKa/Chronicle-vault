# Нижний город
## Описание

Поселение за пределами стен [[Хилсфар]]а, служащее убежищем для всех нелюдей, изгнанных из города согласно Первому закону человечества. Сюда прибыл Эдгар Фрей для расследования случаев появления нечисти.
